# GUT

**G**it **U**I **T**ool

## Install

Using Go

```bash
go install github.com/tsukinoko-kun/gut@latest
```

Using Homebrew

```bash
brew tap tsukinoko-kun/tap
brew install tsukinoko-kun/tap/gut
```
