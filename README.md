# GUT

**G**it **U**I **T**ool

## Install

Using Go

```bash
go install github.com/Frank-Mayer/gut@latest
```

Using Homebrew

```bash
brew tap Frank-Mayer/tap
brew install Frank-Mayer/tap/gut
```
